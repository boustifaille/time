# Time
Time is python file that tell the time remaining to the end of the scool

## Installation
Use git clone
```bash
git clone https://github.com/boustifaille/time.git
```

## Run the file
```bash
cd time
python3 time.py 
```

## Requierements
### pyfiglet 
if you want that the terminal output look prettier
```bash
pip install pyfiglet
```
